# Dossier Images

Placez vos images dans ce dossier selon la structure suivante :

## Structure des images

### Images principales
- `hero-bg.jpg` - Image de fond pour la section Hero (1920x1080px recommandé)
- `hero-main.jpg` - Image principale de la section Hero (800x500px recommandé)
- `history.jpg` - Image pour la section "Notre histoire" (600x600px recommandé)
- `contact-bg.jpg` - Image de fond optionnelle pour la section contact (200x200px recommandé)

### Images des services
Placez les images des services dans le dossier `services/` :
- `services/terrassement.jpg` - Image pour le service Terrassement (400x250px recommandé)
- `services/assainissement.jpg` - Image pour le service Assainissement (400x250px recommandé)
- `services/amenagement.jpg` - Image pour le service Aménagement extérieur (400x250px recommandé)
- `services/enrobe.jpg` - Image pour le service Enrobé (400x250px recommandé)
- `services/benne.jpg` - Image pour le service Benne à gravats (400x250px recommandé)
- `services/pavage.jpg` - Image pour le service Pavage (400x250px recommandé)
- `services/transport.jpg` - Image pour le service Transport (400x250px recommandé)

## Formats recommandés
- Format : JPG ou WebP pour les photos, PNG pour les logos
- Optimisation : Compressez les images avant de les ajouter pour améliorer les performances
- Taille : Les images seront automatiquement redimensionnées par le navigateur, mais des images optimisées améliorent les performances

## Placeholders
Si une image n'est pas trouvée, un placeholder SVG sera automatiquement affiché.
