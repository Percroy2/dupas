# Dossier Galerie

Placez vos images de projets dans ce dossier.

## Structure recommandée

### Images principales
- `terrassement-avant.jpg` - Photo avant travaux de terrassement
- `terrassement-apres.jpg` - Photo après travaux de terrassement
- `assainissement-avant.jpg` - Photo avant installation assainissement
- `assainissement-apres.jpg` - Photo après installation assainissement
- `amenagement-avant.jpg` - Photo avant aménagement
- `amenagement-apres.jpg` - Photo après aménagement
- `enrobe-avant.jpg` - Photo avant pose d'enrobé
- `enrobe-apres.jpg` - Photo après pose d'enrobé
- `pavage-avant.jpg` - Photo avant pavage
- `pavage-apres.jpg` - Photo après pavage
- `lotissement-avant.jpg` - Photo avant travaux lotissement
- `lotissement-apres.jpg` - Photo après travaux lotissement
- `raccordement-avant.jpg` - Photo avant raccordement
- `raccordement-apres.jpg` - Photo après raccordement
- `ecole-avant.jpg` - Photo avant travaux école
- `ecole-apres.jpg` - Photo après travaux école

## Format recommandé
- Format : JPG ou WebP
- Taille : 1200x800px minimum pour de belles images
- Optimisation : Compressez les images avant de les ajouter

## Note
Les images avant/après sont optionnelles. Si elles ne sont pas fournies, seule l'image principale sera affichée dans la galerie.
